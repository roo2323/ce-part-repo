# Phase 관리 체크리스트

## Phase 시작 전 확인
- [ ] 이전 Phase 모든 Task 완료 확인
- [ ] PROMPT_PLAN.md에서 해당 Phase 정보 확인
- [ ] 필요한 선행 산출물 존재 확인
- [ ] 참여 에이전트 목록 파악
- [ ] 예상 소요 시간 확인

## Phase 시작 선언
- [ ] Phase 번호 명시
- [ ] Phase 목표 명시
- [ ] 시작 일시 기록
- [ ] 에이전트별 Task 할당 목록 작성
- [ ] 의존성 순서 명시

## Phase 시작 선언문 템플릿
```markdown
## [ORCHESTRATOR] Phase {N} 시작 선언

**시작 일시**: {YYYY-MM-DD HH:MM}
**Phase 목표**: {목표 설명}

### 작업 할당
| Task | 담당 | 작업 내용 | 선행 작업 |
|------|------|----------|----------|
| {N}-1 | {AGENT} | {설명} | - |
| {N}-2 | {AGENT} | {설명} | {N}-1 |

Phase {N}을 시작합니다.
```

---

## Phase 진행 중 모니터링
- [ ] 일일 진행 상황 확인
- [ ] 블로커 이슈 파악
- [ ] 일정 지연 여부 확인 (1일 이상 → 에스컬레이션)
- [ ] 에이전트 간 의존성 조율

## 진행 상황 업데이트 템플릿
```markdown
## [ORCHESTRATOR] 진행 상황

**Phase**: {N} | **진행률**: {XX}%

| Task | 담당 | 상태 |
|------|------|------|
| {N}-1 | {AGENT} | ✅ 완료 |
| {N}-2 | {AGENT} | 🔄 진행중 ({XX}%) |
| {N}-3 | {AGENT} | ⏳ 대기 |
```

## Phase 완료 확인
- [ ] 모든 Task 완료 및 승인됨
- [ ] 산출물 목록 정리
- [ ] 테스트 통과 확인 (해당 시)
- [ ] 다음 Phase 시작 가능 상태 확인

### 산출물
- {파일 또는 결과물 목록}
