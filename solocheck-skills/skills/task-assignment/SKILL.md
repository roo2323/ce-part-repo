# Task 할당 체크리스트

## 할당 전 확인
- [ ] 선행 Task 완료 여부 확인
- [ ] 대상 에이전트 역할 적합성 확인
- [ ] 필요한 참조 문서 파악
- [ ] 예상 산출물 정의

## 작업 지시서 필수 항목
- [ ] Task ID 명시
- [ ] 대상 에이전트 명시
- [ ] 작업 내용 설명
- [ ] 선행 작업 완료 상태
- [ ] 참조 문서 목록
- [ ] 예상 산출물 목록

## 작업 지시서 템플릿
```markdown
## [ORCHESTRATOR] 작업 지시

**대상**: {AGENT_NAME}
**Task ID**: {Phase}-{Number}
**작업**: {작업 제목}

### 선행 작업
- [x] {완료된 선행 Task}

### 참조 문서
- SPEC.md > {섹션}
- agents/{AGENT}.md > {섹션}
- skills/{skill-name}.md

### 산출물
- {파일 경로 또는 결과물}

### 완료 기준
- [ ] {기준 1}
- [ ] {기준 2}

시작하세요.
```

---

## 할당 후 확인
- [ ] 에이전트가 작업 시작 확인
- [ ] 질문/블로커 발생 시 즉시 대응
- [ ] 예상 시간 내 진행 여부 모니터링
